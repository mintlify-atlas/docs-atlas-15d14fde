---
title: "Password recovery"
description: "How to recover access to your JyPS account"
---

If you have forgotten your password or have been locked out after too many failed login attempts, use the password recovery flow at `/recuperar` to regain access.

## Accessing the recovery page

You can reach the recovery page in two ways:

- **From the login form:** Click the **¿Olvidaste tu contraseña?** link below the password field.
- **From the lockout screen:** After 3 failed login attempts your account is temporarily locked. The blocked-account screen includes a **¿Olvidaste tu contraseña?** button that takes you directly to `/recuperar`.

---

## Recovery flow

The recovery page uses a three-step wizard. Each step is shown in sequence on the same page — you cannot skip ahead.

<Steps>
  <Step title="Enter your institutional email">
    On the first screen, enter the institutional email address associated with your account (format: `usuario@utez.edu.mx`).

    The system validates two conditions before proceeding:

    - The address must be a valid email format (must contain `@`).
    - The address must belong to the `@utez.edu.mx` domain.

    If either condition fails you will see an inline error message and remain on this step. Once the address passes validation, click **Enviar Código** and the system sends a 6-digit verification code to that address.
  </Step>
  <Step title="Enter the verification code">
    The second screen shows six individual digit input boxes. Type the 6-digit code sent to your email — one digit per box. The focus moves to the next box automatically as you type, and pressing Backspace on an empty box moves focus back to the previous one.

    Click **Verificar Código** to validate. If the code is incorrect you will see:

    ```
    Código no válido. Verifique el código enviado a su correo
    ```

    If you need a new code, click **Reenviar código**. This returns you to the email step, clears the digit boxes, and lets you request a fresh code.

    <Note>
      The code is numeric-only. Non-numeric characters are rejected at the input level — each box accepts exactly one digit.
    </Note>
  </Step>
  <Step title="Confirmation">
    After entering a valid code the page transitions to a success screen. A message confirms that a password-reset link has been sent to your email address. Click **Volver al Inicio de Sesión** to return to `/login`.
  </Step>
</Steps>

---

## Navigating back

A **Volver al inicio de sesión** link is displayed at the bottom of the card on both the email step and the code step. It is hidden on the success screen, which shows the **Volver al Inicio de Sesión** button instead.

---

## Troubleshooting

<AccordionGroup>
  <Accordion title="I did not receive the verification code">
    Check your spam or junk folder. The code is sent to the exact address you entered. If you still cannot find it, click **Reenviar código** on the code entry screen to go back and request a new one.
  </Accordion>
  <Accordion title="The system rejects my email address">
    The recovery flow only accepts addresses ending in `@utez.edu.mx`. If your address belongs to a different domain it will not be accepted. Contact the system administrator to verify that your account email is correct.
  </Accordion>
  <Accordion title="My code is not being accepted">
    Make sure you are entering all 6 digits and that you are using the most recently generated code. Each time you click **Reenviar código** a new code is generated and the previous one is no longer valid.
  </Accordion>
</AccordionGroup>
